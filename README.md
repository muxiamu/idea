## 1.分辨率板空间频率计算器  
* 懒得查表，有时候还要自己去换算，不如做一个脚本吧
* 功能：
  * 输入分辨率板的组别(group)和单元(element)，计算其对应的空间频率
  * 和上一条反着来
  * 输入镜头放大倍率和物方空间频率，计算对应的像方空间频率
  * 输入镜头放大倍率和像方空间频率，计算对应的物方空间频率
* USAF 1951分辨率测试板的线对数/毫米
